# Oklahoma General Ledger Data Engineering Project
A repository to explore and analyze the Oklahoma General Ledger dataset, which provides detailed financial information about the state's expenditures and revenues.

## **Data Source**

- **Dataset**: Oklahoma General Ledger
- **Provider**: Oklahoma Office of Management and Enterprise Services (OMES)
- **License**: [CC-BY](https://creativecommons.org/licenses/by/4.0/)
- **URL**: [General Ledger Dataset](https://data.ok.gov/dataset/general-ledger)
